# Outreach Drafts

## Cold Email Template

```
Subject: [Question/hook about their challenge]

Hi [Name],

[HOOK - Show you did research]

I noticed [specific observation about their company].

[BRIDGE - Connect to problem]

Companies like yours often struggle with [problem].

[VALUE - What you offer]

At [Company], we help [audience] [achieve result].

[PROOF - Credibility]

[Customer] saw [result] in [timeframe].

[CTA - Low friction]

Worth a 15-minute call?

[Name]
```

## LinkedIn Templates

### Connection Request
```
Hi [Name],

[Reason for connecting - be specific]
- Enjoyed your post about [topic]
- We're both in [industry/group]
- [Mutual] suggested we connect

Looking forward to connecting,
[Name]
```

### After Connection
```
Hi [Name],

Thanks for connecting!

I noticed [observation]. At [Company], we help [audience] with [problem].

Interested in [offer/resource/conversation]?

[Name]
```

## Follow-Up Sequence

| # | Days | Purpose | Template |
|---|------|---------|----------|
| 1 | 3-4 | Bump | "Floating this back up..." |
| 2 | 7 | Value | Share resource/insight |
| 3 | 14 | Angle | Different approach |
| 4 | 30 | Break-up | "I'll stop reaching out..." |

### Follow-Up #1 (Bump)
```
Hi [Name],

Just floating this back to the top of your inbox.

[Original ask]

Worth a quick chat?

[Name]
```

### Follow-Up #4 (Break-up)
```
Hi [Name],

I haven't heard back, so I'll assume timing isn't right.

I'll stop reaching out, but if [problem] becomes a priority, I'm here.

[Name]
```

## Partnership Proposal

```
Subject: Partnership: [Your Co] x [Their Co]

Hi [Name],

I've been following [Their Company]'s work in [area].

**The Idea:**
[1-2 sentences on partnership concept]

**Why It Works:**
- For you: [Benefit]
- For us: [Benefit]
- For customers: [Benefit]

20-minute call to explore?

[Name]
```

## Response Handling

**Interested:**
```
Great to hear! How does [Day] at [Time] work?
Calendar link: [Link]
```

**Not Now:**
```
Totally understand. Can I check back in [timeframe]?
```

**Not Interested:**
```
Appreciate the response. I'll remove you from follow-ups.
Wishing [Company] all the best.
```

## Best Practices

**Do:**
- Personalize every message
- Keep under 100 words
- One clear CTA
- Follow up (most deals: 5+ touches)

**Don't:**
- Attach files in cold emails
- Use "just checking in"
- Write essays
- Lie about mutual connections
