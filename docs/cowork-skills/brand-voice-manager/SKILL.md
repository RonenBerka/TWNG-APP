# Brand Voice Manager

## Voice Attributes Matrix

Define 3-5 core attributes:

| Attribute | We Are | We Are Not | Example |
|-----------|--------|------------|---------|
| Friendly | Warm, approachable | Overly casual | "We're here to help" vs "Hey dude!" |
| Professional | Expert, reliable | Cold, corporate | "Our expertise..." vs "Per our policy..." |
| Authentic | Genuine, honest | Fake, salesy | "Here's what we found" vs "AMAZING DEAL!" |

## Tone Adaptation

| Context | Tone Shift | Example |
|---------|------------|---------|
| Celebration | Enthusiastic | "We're thrilled to announce..." |
| Apology | Sincere, humble | "We understand this affected you..." |
| Education | Clear, patient | "Here's how it works..." |
| Urgency | Direct | "Action needed: ..." |
| Sales | Confident | "Transform your [X] with..." |

## Style Guide Essentials

### Grammar
| Element | Standard |
|---------|----------|
| Oxford comma | Yes |
| Contractions | Yes ("We're" not "We are") |
| Numbers | Spell 1-9, digits 10+ |
| Currency | Symbol before (₪150, $50) |

### Word Choice
| Use | Don't Use |
|-----|-----------|
| Help | Assist |
| Use | Utilize |
| Buy | Purchase |
| Start | Commence |
| Show | Demonstrate |

## Channel Voice

| Channel | Style |
|---------|-------|
| Website | Full voice, complete sentences |
| Email | Warm, personal, direct |
| Social | Casual, engaging, brief |
| Support | Empathetic, solution-focused |
| Sales | Confident, benefit-driven |

## Voice Audit Checklist

- [ ] Tone matches context
- [ ] Voice attributes present
- [ ] No banned words
- [ ] Active voice
- [ ] Clear and concise
- [ ] On-brand CTAs
