## Response Strategies

| Situation | Response |
|-----------|----------|
| New competitor | Monitor, don't react |
| They launch feature we have | Highlight our depth |
| They launch feature we lack | Emphasize other strengths |
| They raise prices | Value messaging opportunity |
| They lower prices | Avoid price war, emphasize value |
