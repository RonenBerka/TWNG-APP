# TWNG - Guitar Platform Toolkit

## Platform Overview

**Product**: Guitar documentation & history platform
**Stage**: Pre-launch / Beta
**Audience**: Guitar enthusiasts, collectors, luthiers
**Languages**: Hebrew + English
**Tone**: Knowledgeable, passionate, community-focused
