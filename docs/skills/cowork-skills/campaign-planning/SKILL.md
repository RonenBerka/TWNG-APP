# Campaign Planning

## Campaign Brief Template

```
# Campaign: [Name]

## Overview
Type: [Launch/Awareness/Conversion/Retention]
Dates: [Start] - [End]
Budget: ₪[Amount]
Owner: [Name]

## Objectives
Primary: [Measurable goal]
Secondary: [Goal 2], [Goal 3]

## Audience
Primary: [Demographics]
Secondary: [Demographics]
Exclusions: [Who NOT to target]

## Messages
Primary: [Core message]
Supporting: [Message 2], [Message 3]

## Channels
| Channel | Role | Budget % |
|---------|------|----------|
| Meta Ads | Primary | 40% |
| Google | Intent | 30% |
| Email | Nurture | 10% |

## Success Metrics
| Metric | Target |
|--------|--------|
| ROAS | 4x |
| CPA | ₪50 |
| Conv. | 100 |
```

## Timeline Template

```
T-30: Planning (objectives, audience, brief)
T-21: Creative (copy, design, approval)
T-14: Setup (build campaigns, tracking)
T-7: Soft launch (limited, monitor)
T-0: Full launch (scale, optimize)
T+7: Optimization (A/B tests, reports)
```

## Campaign Types

### Product Launch
- Pre: Teaser, list building, influencers
- Launch: Full activation, email blast, PR
- Post: Retargeting, testimonials

### Seasonal/Holiday
- 6-8 weeks: Plan, create
- 4-6 weeks: Early bird, warm up
- 2-4 weeks: Full launch
- During: Flash sales, urgency
- After: Clearance, retargeting

### Always-On
- Awareness: Brand ads, content
- Consideration: Retargeting, nurture
- Conversion: Product ads, cart abandonment
- Retention: Loyalty, cross-sell

## A/B Testing Priority

1. Audiences (who)
2. Offers (what)
3. Creative (how it looks)
4. Landing pages (where)
5. Timing (when)
