## Daily Management Checklist

### Morning Routine (15-20 minutes)

**Meta Ads Manager**:
- [ ] Check total spend vs. daily budget pacing
- [ ] Review ROAS from previous day
- [ ] Identify any campaigns with delivery issues
- [ ] Check for any disapproved ads
- [ ] Monitor frequency (pause if >4-5)
- [ ] Review top 3 performing ad sets
- [ ] Check bottom 3 performing ad sets

**Google Ads**:
- [ ] Review Search Terms Report for new negatives
- [ ] Check Quality Score drops
- [ ] Monitor impression share losses
- [ ] Review Shopping feed errors (if applicable)
- [ ] Check conversion tracking status
- [ ] Review budget pacing

**Quick Actions**:
- Pause ads with ROAS <1.0 after spending 2x target CPA
- Increase budget 10-20% on ad sets with ROAS >4.0
- Add negative keywords from irrelevant searches
- Address any technical issues immediately

### End of Day Review (10 minutes)

- [ ] Final spend check vs. budget
- [ ] Note any anomalies for investigation
- [ ] Prepare next day's tests or changes
- [ ] Update performance tracking sheet

## Weekly Optimization Routine

### Monday: Strategy & Planning

**Review Last Week's Performance**:
1. Pull weekly performance report (both platforms)
2. Calculate key metrics:
   - Total ROAS
   - Average CPA
   - Conversion rate by platform
   - Top performing products
   - Best audience segments

**Plan This Week**:
- Set weekly budget allocation
- Identify 2-3 tests to run
- Plan new creative needs
- Schedule creative refresh if needed

### Wednesday: Mid-Week Optimization

**Meta Performance Review**:
- Analyze ad set performance (3+ day data)
- Reallocate budget to winners
- Pause consistent underperformers
- Check creative fatigue (frequency, engagement drops)
- Review audience overlap (if available)

**Google Ads Tuning**:
- Bid adjustment review
- Update negative keyword lists
- Review search impression share
- Adjust device/location modifiers
- Check Shopping feed quality

### Friday: Creative & Testing

**Creative Performance**:
- Identify top 3 and bottom 3 creatives
- Analyze why winners work (imagery, copy, offer)
- Plan creative refresh for next week
- Review competitor ads (Facebook Ad Library)

**Test Results**:
- Analyze any A/B tests running
- Declare winners (statistical significance)
- Implement learnings
- Plan next round of tests

## Performance Optimization Framework

### When to Optimize vs. When to Wait

**Optimize Immediately If**:
- ROAS <1.0 after spending 2x target CPA
- Frequency >5 on cold audiences
- CTR <0.5% on cold, <1.5% on warm
- CPC 50%+ above account average
- Quality Score <5 in Google Ads
- Disapproved ads or delivery issues

**Wait and Monitor If**:
- Campaign in learning phase (<50 conversions/week)
- Only 1-2 days of data
- ROAS between 1.5-2.5 (acceptable range)
- Testing phase (need full week of data)

### Scaling Successful Campaigns

**Vertical Scaling** (Increase Budget):
- Only scale winners (ROAS >3.5 consistently)
- Increase budget 20% every 3-5 days
- Monitor for performance degradation
- Don't exceed 50% increase per week

**Horizontal Scaling** (Duplicate):
- Duplicate ad set with new audience
- Test in different placements
- Expand to new geographic regions
- Create similar lookalike audiences

**Creative Scaling**:
- More variations of winning concepts
- Test slight modifications
- Seasonal variations
- Product line expansion

### Troubleshooting Common Issues

**Low Delivery / Limited by Budget**:
- Increase daily budget
- Expand audience size
- Broaden targeting
- Check for audience overlap

**High CPA / Low ROAS**:
- Review landing page conversion rate
- Check audience relevance
- Analyze creative effectiveness
- Review offer competitiveness
- Verify pixel tracking

**High Frequency**:
- Expand audience size
- Refresh creative
- Implement frequency caps
- Reduce budget
- Exclude recent converters

**Low CTR**:
- Test new creative concepts
- Improve headline/copy
- Test different offers
- Check audience relevance
- Review ad placement performance

**Disapproved Ads**:
- Review Meta/Google policies
- Check for prohibited content
- Remove claims requiring proof
- Adjust imagery if needed
- Appeal if unjustified

## Testing Framework

### A/B Testing Best Practices

**Test One Variable at a Time**:
- Audience: Interest A vs. Interest B
- Creative: Image style A vs. B
- Copy: Headline variation A vs. B
- Offer: Free shipping vs. 10% off
- Landing page: Layout A vs. B

**Test Structure**:
- Minimum budget: â‚ª200-500 per variation
- Test duration: 5-7 days minimum
- Sample size: 100+ conversions for significance
- Control group: Always include current best performer

**Statistical Significance**:
- Use significance calculator
- Minimum 95% confidence level
- Account for sample size
- Don't call tests too early

### Priority Testing Queue

**High Priority** (Test First):
1. Landing page conversion rate optimization
2. Offer testing (free shipping vs. discount)
3. Top-performing product expansion
4. Audience LAL percentage (1% vs. 3% vs. 5%)
5. Ad creative formats (carousel vs. video)

**Medium Priority**:
1. Copy length variations
2. CTA button text
3. Placement performance
4. Device-specific campaigns
5. Time-of-day bidding

**Low Priority** (Test When Optimized):
1. Minor copy tweaks
2. Emoji usage
3. Subtle color changes
4. Font variations
