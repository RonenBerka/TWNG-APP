##  Budget Buying Tips

### What to Check
| Issue | Solution |
|-------|----------|
| **Fret ends** | File if sharp |
| **Nut slots** | May need adjustment |
| **Setup** | Likely needs proper setup |
| **Electronics** | May need upgrade |
| **Tuners** | Often upgrade candidate |

### Best Value Tiers
| Price | What to Expect |
|-------|----------------|
| **$50-100** | Basic playable, needs work |
| **$100-200** | Decent starter |
| **$200-350** | Good instrument |
| **$350-500** | Very good, upgrade parts |

### Upgrade Priority
1. **Setup** - Most important!
2. **Nut** - Cheap improvement
3. **Tuners** - If slipping
4. **Pickups** - When ready
5. **Electronics** - Pots, switch
