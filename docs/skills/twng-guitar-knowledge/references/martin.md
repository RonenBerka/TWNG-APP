## üè≠ Model Naming System

### Format: [Shape]-[Style Number][Suffix]

**Examples:**
- **D-28** = Dreadnought, Style 28
- **000-18** = Triple-O, Style 18
- **OM-42** = Orchestra Model, Style 42
- **HD-28** = Herringbone Dreadnought, Style 28

## üìÖ Key Eras

### Pre-War (Before 1945) - Golden Era

| Period | Features |
|--------|----------|
| **1833-1867** | New York era |
| **1867-1900** | Early Nazareth |
| **1900-1916** | Concert/parlor dominance |
| **1916** | Dreadnought introduced (for Ditson) |
| **1931** | D-28 introduced |
| **1934** | 14-fret neck standard |
| **1939-1944** | Herringbone discontinued |

**Pre-War Identifiers:**
- Scalloped X-bracing
- Herringbone trim (D-28)
- Pyramid bridge (early)
- 12-fret necks (pre-1934)
- Slotted headstock (some)
- Small pickguard
- Brazilian rosewood

### Post-War (1945-1970)

| Change | Year |
|--------|------|
| Non-scalloped bracing | 1944+ |
| Adjustable truss rod | 1967 |
| Quality still excellent | Throughout |

### Modern Era (1970-Present)

| Period | Features |
|--------|----------|

| 1980s | Vintage reissue program begins |
| 1990s | HD-28 (herringbone returns) |

## ‚ö†Ô∏è Common Identification Mistakes

### 1. Pre-War vs Reissue

**Real Pre-War tells:**
- Serial number in correct range
- Correct label style
- Natural aging (not relic'd)
- Brazilian rosewood (restricted since 1969)
- Correct bracing pattern
- Period tuners (open-back)

### 2. Authentic vs Standard

**Authentic series features:**
- Hide glue construction
- T-bar truss rod (when applicable)
- Exact historical specs
- Higher price point
- Different label

### 3. USA vs Mexico

**X/Road Series:**
- Different serial format
- Made in Mexico label
- HPL back/sides (some)
- Still good quality!

### 4. Style Number Confusion

**Remember:**
- Higher style = more appointments
- Same shape can have many styles
- D-18 ‚â† D-28 ‚â† D-45
