# GTM Strategy Skill

Create comprehensive Go-To-Market strategies for products and services.

## Output Format

Generate GTM strategy as structured markdown. Adapt depth based on product stage and complexity.

## GTM Framework

### 1. Executive Summary
- Product one-liner
- Target market
- Core value proposition
- Launch timeline
- Primary success metric

### 2. Market Analysis

#### Market Sizing
- TAM / SAM / SOM
- Growth rate and trends

#### Competitive Landscape

| Competitor | Positioning | Strengths | Weaknesses | Price |
|------------|-------------|-----------|------------|-------|

#### Market Gaps
- Unmet needs
- Underserved segments

### 3. Target Audience

#### Primary Persona
```
Name:
Role:
Pain Points:
Goals:
Behaviors:
Channels:
```

#### Audience Prioritization
Rank by: accessibility, willingness to pay, strategic value.

### 4. Positioning & Messaging

#### Positioning Statement
```
For [target audience]
Who [need/problem]
[Product] is a [category]
That [key benefit]
Unlike [alternatives]
We [unique differentiator]
```

#### Messaging Hierarchy

| Level | Message | Use Case |
|-------|---------|----------|
| Tagline | 5-7 words | Brand, ads |
| Value Prop | 1 sentence | Homepage |
| Elevator | 30 seconds | Sales, PR |

### 5. Channel Strategy

| Channel | Goal | Cost | Priority |
|---------|------|------|----------|

For each priority channel: tactics, content types, frequency, KPIs.

### 6. Launch Phases

```
Phase 1: Prep (Weeks 1-4)
Phase 2: Soft Launch (Weeks 5-6)
Phase 3: Public Launch (Week 7)
Phase 4: Scale (Weeks 8-12)
```

Include launch checklist and risk mitigation table.

### 8. Success Metrics

#### North Star Metric
Single metric capturing value delivery.

#### KPIs

| Metric | Target | Tracking |
|--------|--------|----------|

### 9. Team & Resources

| Role | Responsibility |
|------|----------------|

## Customization by Product Type

**B2B**: Add sales enablement, ABM, longer cycles, pilot program
**B2C**: Add viral loops, community, influencers, referral program
**Marketplace**: Add supply/demand balance, geographic rollout
**SaaS**: Add freemium strategy, onboarding, retention, pricing tiers

---

## Process

1. Clarify scope (product, market, budget, timeline)
2. Research competitors and audience
3. Draft strategy using framework
4. Review positioning for differentiation
5. Validate metrics are measurable
6. Deliver as markdown or export to docx/pptx
