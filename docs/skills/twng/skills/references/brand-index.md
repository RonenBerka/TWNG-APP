### 6-in-line (All tuners on one side)
- Fender, Squier, G&L
- Ibanez, Jackson, Charvel, ESP, Schecter
- Most superstrats
- Yamaha (many models)

### 3+3 (Three tuners per side)
- Gibson, Epiphone
- PRS
- Gretsch
- Rickenbacker
- Guild
- Martin, Taylor (acoustics)
- Most acoustic guitars

### 4+2
- Music Man / Ernie Ball
- Some Warwick

### Headless
- Steinberger
- Strandberg
- Status
- Some Kiesel
