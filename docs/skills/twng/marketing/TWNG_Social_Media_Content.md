## Platform-Specific Notes

### Instagram:
- Square (1:1) or vertical (4:5) images preferred
- Carousel posts get higher engagement
- Use Stories daily
- Reels for tutorials and behind-the-scenes

### Facebook:
- Longer captions OK
- Link posts work well
- Groups for community building (consider a TWNG group)
- Events feature for launch countdown

### Cross-Posting:
- Adjust captions slightly for each platform
- Instagram: More hashtags, shorter sentences
- Facebook: More conversational, can include links in posts
