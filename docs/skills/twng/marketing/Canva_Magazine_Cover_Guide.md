# TWNG Magazine Cover — Canva Guide

## Step 1: Create New Design

1. פתח Canva
2. לחץ **Create a design**
3. בחר **Custom size**: `1080 × 1350 px` (Instagram Portrait)

---

## Step 2: הוסף תמונת רקע

1. **Upload** → העלה את תמונת הגיטרה
2. גרור לקנבס, הגדל ל-**full bleed** (שיכסה הכל)
3. מרכז את הגיטרה/נגן באזור הימני-מרכזי

---

## Step 3: הוסף Gradient Overlay

1. **Elements** → חפש "gradient"
2. בחר מלבן שחור-שקוף
3. או: **Elements** → **Shapes** → מלבן
4. צבע: **שחור**, Transparency: **40%** למעלה, **80%** למטה
5. מקם על כל התמונה

**Alternative:** הוסף 2 מלבנים:
- מלבן עליון: שחור 30% opacity, blur
- מלבן תחתון: שחור 70% opacity, gradient

---

## Step 4: לוגו TWNG (למעלה במרכז)

1. **Upload** → העלה את הלוגו (PNG שקוף)
2. **או** הקלד טקסט "TWNG":
   - **Font:** Oswald Bold
   - **Size:** 80px
   - **Color:** `#F59E0B` (Amber)
3. מקם: **מרכז למעלה**, כ-40px מהקצה

---

## Step 5: Tagline (מתחת ללוגו)

1. הוסף טקסט: `EVERY GUITAR HAS A STORY`
2. **Font:** Oswald Medium
3. **Size:** 18px
4. **Letter spacing:** 200 (או max)
5. **Color:** `#F59E0B`
6. מקם: מתחת ללוגו, כ-20px רווח

---

## Step 6: Issue Info (פינה ימנית עליונה)

1. הוסף טקסט:
```
ISSUE NO. 001
FREE FOREVER
```
2. **Font:** Oswald Regular
3. **Size:** 12px
4. **Color:** White, 70% opacity
5. **Alignment:** Right
6. מקם: פינה ימנית עליונה

---


---

## Step 8: Main Headline (למטה)

### Guitar Name:
```
FENDER
STRATOCASTER
```
1. **Font:** Playfair Display Black (או Bold)
2. **Size:** 64-72px
3. **Color:** White
4. **Line height:** 0.9 (צפוף)
5. הוסף **Shadow:**
   - X: 0, Y: 4
   - Blur: 12
   - Color: Black 60%

### Subtitle:
```
The One That Started Everything
```
1. **Font:** Oswald Medium
2. **Size:** 24px
3. **Color:** `#F59E0B`
4. **Letter spacing:** 50

**מיקום:** צד שמאל למטה, כ-100px מהקצה התחתון

---

## Step 9: Bottom Tagline

```
"Finally, a place to keep them."
```
1. **Font:** Playfair Display Italic
2. **Size:** 16px
3. **Color:** White, 80% opacity
4. מקם: פינה שמאלית תחתונה, כ-30px מהקצה

---

## Step 10: Final Touches

1. **Effects** על כל הטקסטים:
   - Shadow: subtle drop shadow
   - או: Glow שחור קל לקריאות

2. בדוק שכל הטקסט קריא על התמונה

3. **Export:**
   - PNG
   - Highest quality
