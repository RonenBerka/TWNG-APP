## ‚ö†Ô∏è Common Identification Mistakes

### 1. Jackson vs Charvel

**Jackson tells:**
- More aggressive shapes (V, Kelly)
- "Jackson" headstock
- Metal-focused

**Charvel tells:**
- Strat-based shapes
- "Charvel" headstock
- 80s superstrat heritage

### 2. USA vs Import

**USA Select tells:**
- "Made in USA" clearly marked
- USA Select on headstock
- Premium hardware
- Higher price ($3,000+)

### 3. Era Identification

**1980s originals:**
- Often custom shop quality
- San Dimas address
- Sequential serials
- High collectible value
