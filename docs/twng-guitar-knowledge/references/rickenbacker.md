## üî¢ Serial Number Decoding

### Format (Post-1960)

**Letter + Numbers:**
- Letter = Year
- Numbers = Production sequence

| Letter | Decade | Example |
|--------|--------|---------|
| A-L | 1960s | A1234 = 1961 |
| M-Z | 1970s | M = 1975 |
| A-L | 1980s | Repeats |
| M-Z | 1990s | Repeats |

### Modern (2000+)

- More complex coding
- Contact Rickenbacker for verification
- Includes date codes

### Serial Location

- Jackplate
- Inside body (some)
- Neck heel (some)

## ‚ö†Ô∏è Common Identification Mistakes

### 1. Toaster vs Hi-Gain

**Toaster tells:**
- Chrome slotted top
- Lower height
- 1960s models (and reissues)

**Hi-Gain tells:**
- Rectangular black cover
- Higher output
- 1970s+ standard

### 2. Model Numbers

**Number meanings:**
- First digit: Series (3=semi, 6=solid)
- Second digit: Features (0=standard, 6=deluxe)
- Third digit: Pickup count

### 3. Age Verification

**Due to serial complexity:**
- Contact Rickenbacker directly
- Check feature consistency
- Verify with experts
