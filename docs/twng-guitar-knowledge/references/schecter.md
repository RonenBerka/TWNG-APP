## ‚ö†Ô∏è Common Identification Mistakes

### 1. Diamond Tiers

**Know the difference:**
- Platinum (Korea) = Best import
- Gold (Korea) = Great value
- Silver (Indonesia) = Mid
- Bronze (Indonesia) = Entry

### 2. USA vs Import

**USA tells:**
- "USA" clearly marked
- USA Custom Shop label
- Premium price ($3,000+)
- Custom options

### 3. Series Confusion

**Remember:**
- Hellraiser = EMG active, metal
- Blackjack = Seymour Duncan, rock
- Omen = Entry passive
- Demon = Entry active
