## üî¢ Serial Number Decoding

### Japan (Fujigen) Format

**Format: FYYMMXXX**
- F = Fujigen
- YY = Year (2 digits)
- MM = Month
- XXX = Production number

**Examples:**
| Serial | Decode |
|--------|--------|
| F0512345 | 2005, December |
| F9903001 | 1999, March |

### Older Japan Format

**Format: AYYXXXX** (Letter-Year)
- Letter indicates year (see table)
- Followed by sequential number

| Letter | Year |
|--------|------|
| H | 1987 |
| I | 1988 |
| J | 1989 |
| K | 1990 |
| L | 1991 |
| M | 1992 |
| N | 1993 |
| O | 1994 |
| P | 1995 |
| Q | 1996 |
| R | 1997 |
| S | 1998 |
| - | ... |

### Indonesia Format

Variable formats - often includes:
- Year indicators in first digits
- Factory codes

### Korea Format

- Similar to Indonesia
- K or C prefix common

## ‚ö†Ô∏è Common Identification Mistakes

### 1. Japan vs Indonesia

**Check:**
- Serial prefix (F for Fujigen Japan)
- Back of headstock ("Made in Japan" vs "Made in Indonesia")
- Edge tremolo type (Original Edge = Japan)
- Fret quality

### 2. Prestige vs Premium vs Standard

| Feature | Prestige | Premium | Standard |
|---------|----------|---------|----------|
| **Origin** | Japan | Indonesia | Indonesia |
| **Frets** | Jescar or premium | Premium | Standard |
| **Tremolo** | Edge/Lo-Pro | Edge | Edge III |
| **Finish** | Superior | Good | Good |
| **Price** | $1,500+ | $800-1,200 | $400-700 |

### 3. Edge Tremolo Types

**Don't confuse:**
- Original Edge (Japan, excellent)
- Edge Pro (Japan, excellent)
- Edge III (Indonesia, good but different)
- Licensed Floyd (various)

### 4. GIO vs Standard

**GIO indicators:**
- "GIO" on headstock
- Simpler hardware
- Budget components
- Still decent entry guitars
