## Link Building

### Internal Links
- Every blog post links to relevant serial number tool
- Every famous guitar links to related guitars
- Every page links to signup

### External Link Targets

| Type | Examples | Approach |
|------|----------|----------|
| Guitar blogs | Reverb News, Guitar World | Guest posts, mentions |
| Forums | The Gear Page, My Les Paul | Helpful participation |
| YouTube | Guitar channels | Reviews, mentions |
| Music sites | Ultimate Guitar, Songsterr | Content partnerships |

### Tactics

1. **Resource link building:**
   - Create "best of" lists others want to link to
   - Serial number tools as linkable resources

2. **Guest posting:**
   - Write for guitar blogs
   - Offer unique data/insights

3. **Forum participation:**
   - Be helpful, not promotional
   - Link only when genuinely relevant

4. **PR for famous guitars:**
   - Pitch stories to music publications
   - "We documented the 50 most famous guitars"

## Competitor Analysis

### Who Ranks for Our Keywords?

| Keyword | Current #1 | Gap |
|---------|------------|-----|
| guitar serial number lookup | Fender.com | Brand authority |
| gibson serial number decoder | Gibson.com | Brand authority |
| guitar collection app | Various apps | Low authority sites |
| famous guitars | Wikipedia | Depth of content |

### Opportunity:
- Serial number tools on brand sites are limited
- No dedicated "guitar collection" platform ranking
- Famous guitar content is scattered

### Strategy:
- Beat brand sites with better UX and more data
- Create definitive guitar collection resource
- Build comprehensive famous guitar database
