## Brand Essence

**Tagline**: Every guitar has a past. Make sure it has a future.

**Hebrew Version**: לכל גיטרה יש עבר. וודאו שיש לה גם עתיד.

**Mission**: TWNG נוצר כי לכל גיטרה יש סיפור — ואנחנו כאן כדי להדהד אותו.

**What TWNG Is**:
- רשת חברתית של גיטריסטים ואספני גיטרות
- ארכיון חי לתיעוד היסטוריית גיטרות
- פלטפורמה לחיבור בין אספנים, נגנים ואנשי מקצוע
- מקום שבו גיטרות מקבלות זהות דיגיטלית מאומתת

**What TWNG Is NOT**:
- ❌ לא שוק קנייה/מכירה (לא Reverb)
- ❌ לא פורום דיונים רגיל
- ❌ לא קטלוג יבש של מוצרים

## Voice Characteristics

### 1. Passionate but Professional
- מראים אהבה אמיתית לגיטרות
- שומרים על רצינות ואמינות
- לא "מתלהבים יתר על המידה"

**✅ Good**: "הגיטרה הזו מספרת סיפור של 40 שנה"
**❌ Bad**: "וואו!!! גיטרה מטורפת!!!"

### 2. Personal but Universal
- מדברים כאילו אנחנו חלק מהקהילה (כי אנחנו)
- משתמשים ב"אנחנו" ו"שלנו"
- יוצרים הזדהות

**✅ Good**: "כולנו זוכרים את הגיטרה הראשונה שלנו"
**❌ Bad**: "המשתמשים יכולים לתעד את הגיטרות שלהם"

### 3. Heritage-Focused
- מדגישים היסטוריה וסיפורים
- נותנים ערך לזמן ולמורשת
- מחברים עבר-הווה-עתיד

**✅ Good**: "מי ניגן עליה? באיזה אולפנים היא הקליטה? מה הדרך שעברה?"
**❌ Bad**: "הזינו את הפרטים הטכניים של הגיטרה"

### 4. Authentic & Trustworthy
- מדברים על אמינות ואימות
- מדגישים את ערך התיעוד המקצועי
- לא מבטיחים יותר מדי

## Tone by Context

### UI Copy (אפליקציה/אתר)
- קצר וברור
- פעולה-מונחה
- ידידותי אך מקצועי

```
כותרת דף: "הגיטרות שלי"
CTA ראשי: "תעד את הגיטרה שלך"
ריק state: "עדיין אין גיטרות באוסף. הגיע הזמן להוסיף את הראשונה."
```

### Marketing Copy (שיווק)
- רגשי יותר
- מספר סיפורים
- יוצר FOMO חיובי

```
"הגיטרה שלך עברה הרבה. הגיע הזמן שמישהו ידע על זה."
"אל תתנו להיסטוריה להימחק."
"כל גיטרה ראויה לסיפור. ספרו את שלה."
```

### Professional Communication (לות'יירס)
- מקצועי ומכבד
- מדגיש ערך הדדי
- פחות רגשי, יותר עסקי

```
"TWNG מאפשרת לך לתעד את העבודה שלך ולבנות מוניטין מקצועי."
"הוסף חותם אותנטיקציה לגיטרות שטיפלת בהן."
```

### Social Media
- יותר קז'ואל
- שימוש באימוג'י במידה (🎸 🎵 ✨)
- שואל שאלות ומעודד engagement

```
"מה הגיטרה הראשונה שלכם? 🎸"
"הגיטרה הזו עברה 4 בעלים ו-3 יבשות. מה הסיפור שלה?"
```

## Key Messages by Audience

### For Collectors (אספנים)
- **Pain**: מידע מפוזר, אין תיעוד מסודר, קשה להוכיח מקוריות
- **Solution**: תיעוד מקצועי במקום אחד, אימות מלות'יירס

```
"כל המידע על האוסף שלך — במקום אחד, מתועד ומאובטח."
"גיטרה מתועדת היטב שווה יותר. פשוט ככה."
```

### For Musicians (נגנים)
- **Pain**: לא יודעים את ההיסטוריה של הכלי, קשה למצוא טכנאים טובים
- **Solution**: חיבור לקהילה, גישה ללות'יירס מוסמכים

```
"הכר את הגיטרה שלך. היא מכירה אותך."
"מצא את הטכנאי הנכון לגיטרה שלך."
```

### For Luthiers (לות'יירס)
- **Pain**: קשה לבנות מוניטין, אין פלטפורמה מקצועית
- **Solution**: חשיפה ללקוחות, תיעוד עבודה, בניית מוניטין

```
"העבודה שלך מדברת בעד עצמה. עכשיו היא גם נשארת."
"בנה פורטפוליו דיגיטלי של כל גיטרה שטיפלת בה."
```

## Vocabulary

### Words We Use
| English | Hebrew | Context |
|---------|--------|---------|
| Document | תעד | Primary CTA |
| Story | סיפור | Emotional connection |
| Heritage | מורשת | Value proposition |
| Verify | אמת | Trust building |
| Collection | אוסף | User's guitars |
| Journey | מסע | Guitar's history |
| Resonate | להדהד | Brand mission |

### Words We Avoid
| Avoid | Use Instead | Why |
|-------|-------------|-----|
| Buy/Sell | Document/Share | We're not a marketplace |
| Catalog | Archive/Collection | Too cold |
| Database | Living archive | More emotional |
| Users | Members/Collectors | Community feel |
| Post | Share/Document | More meaningful |

## Formatting Guidelines

### Headlines
- Short and punchy (5-8 words)
- Often a question or statement
- Hebrew: right-to-left, no punctuation at end of titles

### Body Copy
- Paragraphs: 2-3 sentences max
- Use bullet points for features
- Break up long text with subheadings

### CTAs
- Action verbs: "תעד", "גלה", "הצטרף", "שתף"
- Clear value: "תעד את הגיטרה שלך" not "לחץ כאן"

## Brand Don'ts

❌ **Don't** use overly salesy language
❌ **Don't** promise features we don't have yet
❌ **Don't** compare directly to competitors
❌ **Don't** use technical jargon without explanation
❌ **Don't** forget the emotional connection to guitars
❌ **Don't** make it sound like just another social network
❌ **Don't** neglect the verification/trust angle

## Quick Reference

**When in doubt, ask:**
1. Does this sound like something a guitar lover would say?
2. Does it emphasize story/heritage?
3. Is it authentic, not salesy?
4. Would I want to read this?

**The TWNG Voice in One Sentence:**
"We speak like passionate guitar collectors who happen to be building a platform for other passionate guitar collectors."
