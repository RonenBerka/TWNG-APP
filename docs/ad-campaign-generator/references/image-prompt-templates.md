# AI Image Prompt Templates

Ready-to-use templates for Midjourney, DALL-E, Gemini, Nano Banana, and other AI image generators.

---

## Universal Prompt Structure

```
[Subject] + [Action/Pose] + [Setting] + [Lighting] + [Mood] + [Style] + [Technical]
```

**Technical elements to always include:**
- Aspect ratio
- Style (photo-realistic, illustration, 3D, etc.)
- "No text" or "No watermarks" if needed
- Quality keywords (8K, professional, sharp focus)

## Specific Industries

### Music/Instruments — GUITAR ACCURACY MANDATORY

**Guitar collectors notice every detail. Inaccurate guitars destroy credibility.**

**ALWAYS specify these elements for guitars:**

| Element | Details to Include |
|---------|-------------------|
| **Inlays** | "Inlays at frets 3-5-7-9-12" (standard dot pattern) |
| **Headstock** | Fender: 6-in-line / Gibson: 3+3 / PRS: 3+3 curved |
| **Body shape** | Stratocaster (double cutaway), Les Paul (single cutaway), Telecaster (slab body) |
| **Pickups** | Strat: 3 single-coils / Les Paul: 2 humbuckers / Tele: 2 single-coils |
| **Bridge** | Strat: tremolo / Les Paul: Tune-o-matic / Acoustic: pin bridge |
| **Hardware** | Chrome, gold, nickel, or black — match the era/model |

**Template:**
```
[Guitar brand and model] with [headstock style] headstock, [body shape] body,
[pickup configuration], [bridge type], [hardware color] hardware,
[finish color/type], inlays at frets 3-5-7-9-12,
[position/context], [setting description], [lighting style],
[mood] atmosphere, [color palette], [photography style], [aspect ratio]
```

**Example — Accurate Fender Stratocaster:**
```
Fender Stratocaster with 6-in-line headstock, double cutaway body,
three single-coil pickups, synchronized tremolo bridge, chrome hardware,
sunburst finish, inlays at frets 3-5-7-9-12,
leaning against vintage Fender amp in home studio,
warm afternoon window light, nostalgic mood, warm earth tones,
documentary photography style, aspect ratio 4:5
```

**Example — Accurate Gibson Les Paul:**
```
Gibson Les Paul with 3+3 headstock, single cutaway body,
two humbucker pickups, Tune-o-matic bridge with stopbar tailpiece,
gold hardware, cherry sunburst finish, trapezoid inlays,
resting on black velvet in guitar case,
dramatic side lighting, luxurious mood, rich warm tones,
product photography style, aspect ratio 4:5
```

**We are dealing with guitar memories — accuracy is non-negotiable.**
