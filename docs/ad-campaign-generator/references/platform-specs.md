# Platform Specifications

Complete specifications for all major ad platforms.

---

## Meta (Facebook & Instagram)

### Image Sizes

| Placement | Size | Ratio |
|-----------|------|-------|
| Feed | 1080x1350 | 4:5 |
| Feed (square) | 1080x1080 | 1:1 |
| Stories/Reels | 1080x1920 | 9:16 |
| Carousel | 1080x1080 | 1:1 |
| Right column | 1200x628 | 1.91:1 |

### Video Specs

| Placement | Size | Duration | File Size |
|-----------|------|----------|-----------|
| Feed | 1080x1350 | 1-240 sec | 4GB max |
| Stories | 1080x1920 | 1-120 sec | 4GB max |
| Reels | 1080x1920 | 15-90 sec | 4GB max |

### Character Limits

| Element | Limit | Recommended |
|---------|-------|-------------|
| Primary text | 125 (before truncation) | 80-100 |
| Headline | 40 | 27 |
| Description | 30 | 27 |
| Link description | 30 | 20 |

### Best Practices

- Mobile-first: 98% of users on mobile
- Text on image: <20% for best delivery
- Video: First 3 seconds critical
- Carousel: 3-5 cards optimal
- UGC-style outperforms polished

## TikTok

### Video Specs

| Element | Spec |
|---------|------|
| Size | 1080x1920 (9:16) |
| Duration | 5-60 sec (21-34 optimal) |
| File size | 500MB max |
| Format | MP4, MOV |

### Character Limits

| Element | Limit |
|---------|-------|
| Ad text | 100 chars (80 recommended) |
| Display name | 40 chars |

### Best Practices

- Hook in first 1 second
- Native, not polished
- Music/sound crucial
- Text overlays for silent viewing
- Vertical only, no letterboxing

## Email

### Image Guidelines

| Element | Recommendation |
|---------|----------------|
| Width | 600px (max 800px) |
| Hero image | 600x300 |
| File size | <200KB per image |
| Format | JPG, PNG, GIF |

### Copy Guidelines

| Element | Optimal | Max |
|---------|---------|-----|
| Subject line | 40 chars | 60 |
| Preview text | 50 chars | 100 |
| Body | 50-125 words | 200 |
| CTA button | 2-4 words | 5 |

### Best Practices

- Single column for mobile
- Button > text link
- Alt text for all images
- Test across clients (Gmail, Outlook, Apple Mail)
